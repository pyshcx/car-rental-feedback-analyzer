Technical Approach

  • TextBlob library for natural language processing
  • Rule-based sentiment classification
  • Keyword matching for issue detection
  • Statistical analysis of review patterns

Code Structure

  • sentiment_analyzer.py: Core analysis engine
  • streamlit_app.py: Web interface
  • Data processing: CSV file handling
  • Visualization: Chart generation

Sentiment Analysis Algorithm

1
Text Preprocessing

Clean and normalize the input text, remove special characters

2
Sentiment Scoring

Calculate polarity (-1 to +1) and subjectivity (0 to 1) scores

3
Classification

Classify as Positive (>0.1), Negative (<-0.1), or Neutral

4
Issue Detection

Identify specific problems using keyword matching
